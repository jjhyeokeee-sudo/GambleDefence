# 프로젝트 규칙 - CardRoguelikeDefense (GambleDefence)

## 프로젝트 기본 정보
- Unity 버전: 2022.3 LTS 또는 Unity 6 (URP 2D)
- 게임 장르: 로그라이크 디펜스 (카드 기반)
- 핵심 루프: 캐릭터 소환 → 카드 뽑기 → 카드 부여 → 디펜스 → 다음 스테이지
- 카드 뽑기에서 나오는 카드들은 플레이어가 수집 가능 -- 각 게임별 보상체계 존재,그 안에서 카드와 같은 강화와 관련된 아이템들을 얻고 그 아이템들이 게임 내에서 랜덤하게 나오면서 캐릭터 강화하는 구조.

## 기술 스택 및 스타일
- 2D URP 사용
- ScriptableObject를 적극 활용 (카드 데이터, 능력치, 아이템 등)
- 모든 클래스명은 영어 (PascalCase)
- 중요한 부분은 한국어로 주석 작성 OK
- 코드 가독성을 최우선으로, 불필요한 복잡함 피함

## Claude가 지켜야 할 규칙
- MCP 도구(UnityMCP)를 최대한 활용해서 씬, GameObject, 컴포넌트 등을 직접 조작
- 코드만 작성하는 대신 가능하면 MCP로 바로 적용
- 큰 변경 전후로 현재 Hierarchy와 Scene 상태를 확인
